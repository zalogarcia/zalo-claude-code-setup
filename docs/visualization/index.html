This is your daily workflow.

One terminal orchestrates with /plan. A second terminal does the work with /autopilot. Then back to the first to verify and merge.

Watch how the handoff actually runs — 8 steps, about 30 seconds.

Two terminals, two commands.

Install once:

git clone https://github.com/zalogarcia/zalo-claude-code-setup ~/.claude-setup
cd ~/.claude-setup && ./install.sh

Then in your daily flow:

Terminal 1 — orchestrator
/plan add a dark mode toggle to my app

# /plan finishes and hands you an /autopilot command. Copy it.

Terminal 2 — worker
/autopilot ...

# When /autopilot finishes, return to Terminal 1 to verify and merge.
