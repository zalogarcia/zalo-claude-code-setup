Autoloop Architecture

What is Autoloop?

Autoloop lets an AI agent autonomously improve your codebase or test your live app. You give it a goal (a briefing), walk away, and it works in a loop: hypothesize → modify → test → commit or revert.

How it works

1. You write a briefing (markdown file describing the goal)

2. Launch the harness (autoloop-harness.sh)

3. The AI works through phases: Briefing → Recon → Sandbox → Integration → Hardening → Report

4. Ratchet pattern: score can only go up, failed experiments are reverted

5. This dashboard monitors everything via .autoloop/ directory

Write a briefing, launch the harness, and the AI works through your task while this dashboard shows real-time progress.
